Results API

The results are available at the following path:

Results for a single country (json)

  /result/{country}.json
  eg. GET /result/au.json

The structure of the result is self-explanatory.

Voting API

A vote can be cast using a POST (application/x-www-form-urlencoded) to the following path:

Parameter Description
country IS0 3166 country code (lowercase) eg. au
party Republican | Democratic

An HTTP status code 200 indicates a successful vote. Use country=test for testing.

Country codes are in accordance with ISO 3166-1 alpha 2 (in English). For "the world excluding US" use "all".
