Results API

The results are available at the following paths:

Results for a single country (json)

  /results/{country}.json
  eg. /results/au.json

Results for multiple countries (json)

  /results?countries={countries}.json
  eg. /results/au+nz.json

Country codes are in accordance with ISO 3166-1 alpha 2 (in English). For "the world excluding US" use "all".
